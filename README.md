# PDF to Encoded Images Module

This project is a simple implementation of a module that converts PDF pages into images and encodes them.

The module supports different encoding types and compression modes, and it can control the maximum total size of the generated images

## Features

- Convert PDF pages into images
- Encode images (Base64 or JPEG)
- Compression modes:
  - Off
  - On
  - Custom
- Maximum total image size control

## Project Structure

pdf_encoder/ → core implementation  
tests/ → test files  
main.py → entry point
